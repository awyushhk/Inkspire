# Inkspire | The Editorial Portfolio Platform

**Inkspire** is a high-end, narrative-driven portfolio builder designed for creatives who view the web as a canvas, not just a container. It prioritizes "Magazine" aesthetics—sharp typography, soft lavender palettes, and sophisticated 2D motion, to make stories feel deeper and more premium.

## 📌 Note

This project is a frontend-focused concept showcasing the homepage of Inkspire. It showcases UI/UX depth and visual storytelling, and is not intended as a complete product or backend-driven application.

## 🚀 Tech Stack
- **Next.js 15** (App Router)
- **Tailwind CSS 4**
- **Framer Motion** (Parallax & Reveal Animations)
- **Lenis** (Smooth Kinetic Scroll)

## 🎨 Key Features
- **Editorial Design**: A "Lilac & Ink" palette focused on white space and sharp serifs.
- **Interactive Motion**: Sticky fanning hero images, staggered masonry showcase, and custom reactive cursor.
- **Responsive & Smooth**: Integrated Lenis scroll for a heavy, premium feel.

## 🛠️ Setup
```bash
npm install
npm run dev
```
